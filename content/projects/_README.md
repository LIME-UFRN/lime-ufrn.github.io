# Projects Folder -- Instructions (Revised Standard)

This folder contains research, network, mobility, and funded projects
coordinated or participated in by LIME.

The structure below is mandatory to ensure consistency, readability, and
long-term maintainability.

------------------------------------------------------------------------

## 1. Filename Convention

YYYY-project-slug.md

Rules: - Lowercase only - Words separated by hyphens - No accents - No
special characters

Example: 2024-brafitec-student-mobility.md

------------------------------------------------------------------------

## 2. Required Metadata (Top of File)

type: Research \| Research Network \| Student Mobility \| Infrastructure
\| Industrial Collaboration period: YYYY--present \| YYYY--YYYY role:
coordinator \| member \| partner

Notes: - Do NOT include "status" or "nature". - Period replaces
status. - Multiple types may be separated by semicolon.

Example:

type: Research Network; Student Mobility period: 2024--present role:
coordinator

------------------------------------------------------------------------

## 3. Content Structure

### Project Title

Short executive description (2--3 lines maximum).

Type: (repeat readable version if desired) Period: 2024--present
Funding: Agency --- Program (Grant number) Website: Full explicit URL
(if available)

#### Institutions

Small lists: comma-separated in one line. Large lists: use bullet
points.

#### Partners -- Industry (if applicable)

-   Company A
-   Company B

#### Partners -- International (if applicable)

-   Institution A
-   Institution B

#### Members (optional)

-   Name (Role)

------------------------------------------------------------------------

## 4. Design Rules

-   Do NOT use horizontal lines inside a project.
-   Keep executive description concise.
-   Use explicit URLs (no hidden embedded links).
-   Use comma-separated institutions only for short lists.
-   Avoid bureaucratic terminology (e.g., "Nature").
-   Keep tone academic and institutional.

------------------------------------------------------------------------
