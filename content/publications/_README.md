# Publications Folder -- Instructions

This folder contains peer-reviewed journal papers, conference papers,
book chapters, and technical articles.

## 1. Filename Convention

YYYY-short-descriptive-slug.md - Lowercase only - Words separated by
hyphens - No accents or special characters

Example: 2026-esp32-edge-ai-energy-forecasting.md

## 2. Required Metadata

First line MUST be: year: YYYY

## 3. Content Structure

### Paper Title

Author1; Author2; Author3

*Venue Name*, Year.

DOI: https://doi.org/xxxx

## 4. Template

year: 2026

### Title of the Paper

Author A; Author B; Author C

*Journal or Conference Name*, 2026.

DOI: https://doi.org/xxxxx
